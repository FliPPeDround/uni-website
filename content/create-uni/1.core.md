## ⚙️ 配置选项

### 交互式配置

::stack
::div{class="p-3 text-lg font-bold"}
  运行命令时，你可以通过添加参数`--gui`来启动图形化界面选择配置
::

```bash
npm create uni-app@latest --gui
```
:read-more{title="图形化界面" to="/create-uni/gui"}
::

### 命令行参数

```bash
npm create uni-app@latest my-app -t vitesse
```


| 选项 | 参数 | 简写 | 说明 |
|------|------|------|------|
| **模板** | `--template` | `-t` | 选择项目模板 |
| **TypeScript** | `--typescript` | `--ts` | 启用 TypeScript |
| **ESLint** | `--eslint` | `-e` | 集成 ESLint 代码规范 |
| **插件** | `--plugin` | `-p` | 添加开发插件 |
| **模块** | `--module` | `-m` | 集成功能模块 |
| **UI库** | `--ui` | `-u` | 选择 UI 组件库 |
| **GUI界面** | `--gui` | - | 启动图形化界面 |
| **信息** | `--info` | - | 显示项目信息 |
